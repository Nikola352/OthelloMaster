#  Othello Master

Othello game built with Python and PyQT5.
Supports 2 players and 1 player vs AI mode.
Minimax algorithm with custom heuristics is used for AI.

## Requirements
<ul>
    <li>Python 3.6+</li>
    <li> PyQT5</li>
</ul>

```
pip3 install pyqt5
```

## How to run
Simply run the main.py file with Python 3.6+.
```
python3 main.py
```
